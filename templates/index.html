Titanic Survival Prediction

Why Titanic Dataset?

A historic dataset used to predict passenger survival using machine learning.

What is This App?

This app analyzes data from the Titanic disaster and uses machine learning to predict survival outcomes.

📊 Features used:
  • Passenger Class (Pclass)
  • Sex & Age
  • Fare
  • Relatives Aboard
